Projects

Here you can find some of my most relevant projects developed so far.

Motonitor Hack UPC (2022)

This project makes the calculation based on a basic Machine Learning Model that takes data from different motorbikes (previously extracted from second-hand motorbike online shops with our tool Motonitor-Deep-Scrapper) and uses it to make a learning model that will predict with a pretty good accuracy (82% scored) an estimated price for the motorcycle.

The development of this project was driven by the challenge presented by Mundimoto during the HackUPC event that took place in the year 2022.

Instagram comment classification based on sentiment analysis (2023)

This project is a part of my bachelor thesis is a study on how to classify the polarity of offensive comments on social networks, specifically on Instagram and oriented to professional football players around the world.

This project is composed by multiple jupyter notebooks files that make possible the classification of the comments and in each of them a part of the pipeline of the creation of a model from scratch is carried out. You can find the complete project in the title link and read the "memory.pdf" file with more detailed info about all the process and the tools used.

About jorgec reflex app
